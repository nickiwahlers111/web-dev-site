InforMetro

This is a ETL data pipeline project for a Data Engineering course at PSU. The relevant data for this project was breadcrumb data for all busses in the C-Tran metro system in the SW Washington and Portland Metro areas. My teammates and I collected this data once per day. To collect the data, we used Apache Kafka consumers to read the data from the source, produced the data from that topic with a Kafka producer, validated and transformed the data, and read into database tables so that we could run analysis on the data. For example, we could map every bus that routed through PDX.

Technologies
  • PostgreSQL
  • Apache Kafka
  • Pandas
  • Mapbox API
  • Python

Music Hole

This is a project for Front end Web Development. The Music Hole app allows users to search musical artists and get other artist recommendations. The user can then fall down the music hole and continue to get recommendations by clicking on the results of each search.
